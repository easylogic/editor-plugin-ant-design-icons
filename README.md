# Vue Plugin Template for EasyLogic Studio 

## Features 

* initialize plugin 
* add UI Element 
* add MenuItem in sidebar 
* add Inspector 
* add Html Renderer 
* add Layer Model 

# Local Dev 

Please install `vue`

```
npm install
npm install @easylogic/editor vue
npm run dev 
```

# How to run

```js
npm run dev 
```
