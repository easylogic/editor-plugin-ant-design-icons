# AntDesign Icons Plugin for EasyLogic Studio 

This plugin makes ant design icons layer.

## Features 

* run new component 
* add BaseProperty in library 

# Local Dev 

```
npm install
npm install -D @easylogic/editor @ant-design/icons-svg
npm run dev 
```

# How to run

```js
npm run dev 
```

# How to build 

```js
npm run build
```
